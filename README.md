# ScoreDock Website ⚽️

This is the official landing page for **ScoreDock**, a native macOS menu bar/dock application for tracking live sports scores. 

## 🚀 Live Site
*(Add your GitHub Pages URL here once deployed, e.g., `https://[your-username].github.io/scoredock-website`)*

## 🛠️ Technology Stack
This landing page is built to be extremely lightweight and blazing fast.
* **HTML5**: Semantic structure.
* **Vanilla CSS**: Custom styling, Apple-style glassmorphism, and responsive layout.
* **No Build Tools**: Zero dependencies. It's ready to be deployed anywhere instantly.

## 💻 Local Development
Since this project uses plain HTML and CSS, you don't need any complex servers to run it.

1. Clone the repository:
```bash
git clone https://github.com/yourusername/scoredock-website.git
```
2. Navigate to the folder:
```bash
cd scoredock-website
```
3. Open `index.html` in your favorite browser! 

*(Optional)* If you want to run a local web server for testing:
```bash
python3 -m http.server
```

## 📦 Deploying
This project is designed to be hosted for free using **GitHub Pages**. Simply push this code to a public GitHub repository and enable GitHub Pages in the repository settings to deploy instantly.
